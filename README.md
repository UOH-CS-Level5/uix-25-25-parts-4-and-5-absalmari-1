# Personal Supervisor System – UI/UX Portfolio Project

## Overview
This repository contains a Personal Supervisor System developed as part of the final project for:

551460_A25_T2: User Interfaces and Experience – Individual UI/UX Portfolio :contentReference[oaicite:0]{index=0}

The project focuses on designing and implementing a UI/UX solution to support student wellbeing and engagement.

---

## Project Context

Based on Appendix A (DDD Brief), the system provides a digital solution to help monitor and improve interactions between:

- Students  
- Personal Supervisors (PS)  
- Senior Tutors (ST)  

---

## Key Features

- Students self-report wellbeing and progress  
- PS review student engagement  
- Two-way meeting booking (Student ↔ PS)  
- ST monitors overall system activity

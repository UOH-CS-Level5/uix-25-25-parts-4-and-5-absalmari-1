# Part 5 – Testing

## Overview
This folder contains all materials related to **Part 5: User Testing and Evaluation** of the Personal Supervisor System.

The purpose of this section is to demonstrate how the application was evaluated using both **qualitative** and **quantitative** usability testing methods, along with supporting evidence and results.

---

## Contents

### 1. Participant Consent Form
- **File:** `Participant Consent Form.docx`  
- A signed consent form completed by the participant who took part in the usability testing.  
- Ensures ethical considerations were followed during testing.

---

### 2. Qualitative Testing – Feedback
- **File:** `Feedback.html`  
- A feedback-based usability test used to gather user opinions, observations, and suggestions about the interface.

- **Results:** `feedback_results.txt`  
- Contains the collected responses and insights from the participant.

---

### 3. Quantitative Testing – System Usability Scale (SUS)
- **File:** `sus.html`  
- A standardized usability questionnaire used to measure system usability.

- **Results:** `sus_results.txt`  
- Contains the calculated SUS score and responses from the participant.

## Summary

This folder demonstrates:
- Application of both **qualitative and quantitative testing methods**  
- Ethical testing practices through participant consent  
- Collection and analysis of usability data  
- Use of external tools (Figma and Lyssna) to support evaluation  

These materials support the improvements made in the **After Adjustments** version of the application.
